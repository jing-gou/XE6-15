- VoiceLife 声活

---

## 问题
- 牙膏。下班。忘了。
- 脑子记不住小事，App 操作太繁琐。
- 不是缺工具，是工具都假设你有手、有屏幕、有注意力。

---

## 为谁做
- 不想每次掏手机的人
- 通勤、做饭、走路中的人
- 无屏设备使用者

---

## 现有做法与不足
| 做法 | 不足 |
| --- | --- |
| 手机闹钟 | 只响铃，不知道什么事 |
| Siri / 小爱 | 创建用了语音，提醒还是 GUI |
| 日历 App | 创建靠打字，修改更麻烦 |
| 微信传输助手 | 自己记、自己翻、自己管 |

- 机会：没人把「创建 → 提醒 → 响应 → 修改」用语音串起来。

---

## VoiceLife 怎么做
口诀：说出来 → 到点听 → 随口改

---

## 第一步：说出来
- 按住说话，松开记好。
- "下班买牙膏" → 记录下来
- "明早七点吃药，每天" → 周期提醒
- "今天有哪些安排？" → 即时回答

---

## 第二步：到点听
- 预告（提前 15min）：短铃 + 语音
- 提醒（到时）：语音播报 + 持续铃声
- 1 分钟无回应 → 自动推迟

四件事，一口回应：
- "知道了" → 结束
- "十分钟后再提醒" → 推迟
- "会议在哪？" → 追问详情
- "跳舞推迟，其余知道了" → 精准边界

---

## 第三步：随口改
| 操作 | 例子 |
| --- | --- |
| 修改单次 | "下午会改到四点半" |
| 仅改本次 | "周五跑步改周六" |
| 永久改 | "以后都改周六" |
| 跳过 | "周五不去了" |
| 删除 | "删掉每周一早会" |
| 撤销 | "不对，撤销刚才的" — 15min 内有效 |
